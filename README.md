## Web Interface

You can also use Conspectum via web interface:

1. Install dependencies: `pip install -r requirements.txt`
2. Set environment variables in `.env` (same as for bot)
3. Run web server: `python -m src.web`
4. Open http://localhost:8000 in your browser

## Обзор

**Конспектум** — сервис для автоматической генерации полноценных конспектов по аудиозаписям лекций. Пользователь загружает аудио файл, система транскрибирует речь в текст, обогащает его информацией из внешних источников, структурирует и оформляет в формате LaTeX с формулами, определениями и ссылками на литературу.

## Проблема

Студенты и преподаватели ежедневно сталкиваются с большим объёмом лекционного материала. Во время лекции сложно одновременно слушать, понимать и записывать — что-то неизбежно теряется. После занятия остаётся аудиозапись, но переслушивать её целиком и вручную оформлять конспект — долго и утомительно. Существующие инструменты транскрибации выдают «сырой» текст без структуры, формул и дополнительного контекста, что делает его малопригодным для повторения и подготовки к экзаменам.

## Решение

Конспектум принимает на вход аудиозапись лекции и превращает её в готовый к использованию конспект:

1. **Транскрибация** — автоматическое распознавание речи из загруженного аудиофайла и преобразование в текст.
2. **Структурирование** — выделение разделов, подразделов, ключевых определений и теорем.
3. **Обогащение** — автоматический поиск и добавление релевантной информации из внешних источников (электронные учебники, статьи, базы знаний).
4. **Формализация** — распознавание и корректное оформление математических формул в LaTeX.
5. **Оформление** — генерация итогового документа в формате LaTeX с оглавлением, нумерацией и библиографическими ссылками.
6. **Мультиязычность** — поддержка генерации конспектов на русском и английском языках.

## Функциональность (MVP)

- Загрузка аудиофайла
- Автоматическая транскрибация речи в текст
- Генерация структурированного конспекта с разделами и подразделами
- Вставка математических формул в формате LaTeX
- Обогащение конспекта информацией из внешних источников
- Добавление библиографических ссылок
- Выбор языка конспекта (русский / английский)
- Скачивание результата в формате `.tex` и `.pdf`

## Как это работает

- Модель [gpt-4o-audio-preview](https://developers.openai.com/api/docs/models/gpt-4o-audio-preview) от OpenAI — основа сервиса, используется для генерации текста по аудио.
- Модель [gpt-4.1](https://developers.openai.com/api/docs/models/gpt-4.1) от OpenAI для постобработки текста конспекта.
- Пакет [pdflatex](https://pypi.org/project/pdflatex/) для работы с LaTeX и формирования итогового конспекта в формате PDF.
- Библиотека [pydub](https://pypi.org/project/pydub/) для предобработки аудио.

## Начало работы

1. Откройте Telegram-бота по ссылке: [@grom_conspectum_bot](https://t.me/grom_conspectum_bot)
2. Выберите язык конспекта (опционально)
3. Отправьте боту `.wav`-файл с записью лекции.
4. Дождитесь завершения обработки — бот пришлёт готовый конспект в форматах `.tex` и `.pdf`.

## Локальная разработка

```
# Склонируйте репозиторий
git clone https://github.com/TTPO100AJIEX/Conspectum

# Перейдите в директорию проекта
cd Conspectum

# Установите зависимости
pip install -r requirements.txt

# Укажите переменные окружения в файле src/.env по аналогии с .env.example
# Запустите программу
cd src
python -m bot
```

## Дальнейшие шаги

- В дальнейшем сервис планируется расширить за счет поддержки видеофайлов, чтобы пользователь мог загружать не только аудиозаписи, но и видеолекции без предварительного извлечения звука вручную.

- Отдельным направлением развития может стать live-формат, при котором сервис будет работать с лекцией в реальном времени и формировать конспект по ходу выступления или сразу после его завершения.
